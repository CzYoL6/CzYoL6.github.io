二叉搜索树

什么是二叉搜索树

二叉搜索树的每个结点都是一个对象，除了key 和卫星数据外，每个结点还包括left和right和p，分别指向左孩子，右孩子。

二叉搜索树的性质

  • 左子树中的结点均$\leq$根节点，右子树中的结点均$\geq$根节点。

  • 一颗BST的中序遍历序列就是排好序的递增序列。

  • 遍历一颗有n个结点子树，需要$\theta(n)$的时间。

    证明：

    因为要遍历全部n个结点，所以$T(n)=\Omega(n)$。
    对于一颗空树，需要耗费一个小常数时间c用于测试$x\neq null$。

    否则假设左子树有k个结点，右子树有n-k-1个结点，则$T(n) \leq T(k) + T(n - k - 1) + d$，$d > 0$。由替换法，假设$T(n) \leq (c+d)n + c$，那么可以证得$T(n) = O(n)$。$n = 0$，显然成立；$n > 0$时，

    $T(n) \leq T(k) + T(n - k - 1) + d = ((c + d)k + c) + ((c + d)(n - k - 1) + c) + d = (c + d)n + c - (c + d) + c + d = (c + d)n + c$

    即$T(n) = O(n)$

    由此得$T(n) = \theta(n)$

  • 删除节点为叶节点：直接删除

  • 删除节点有左子树，右子树为空：删除结点，并用左子树代替

    删除节点有右子树，左子树为空：删除节点，并用右子树代替
